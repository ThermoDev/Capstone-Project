Running the Frontend

Requirements:
    - npm

Instructions:
    1. Navigate to the frontend/ folder.
    2. Run the command "npm install" to install the frontend dependencies.
    3. Run the command "npm run build" to create a production-ready build of the application.
    4. Run the command "npm run start" to start the application.


Running the Backend

Requirements:
    - pip
    - Python 3.7

Instructions:
    1. Navigate to the backend/ folder.
    2. Run the command "pip install -e ." to install the backend application as a pip package.
    3. Run the command "flask run" to run the backend application.
